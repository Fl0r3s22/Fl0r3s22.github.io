Intorducción al lenguaje de marcado HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear y estructurar páginas web. Proporciona la estructura y el contenido de una página web, definiendo los elementos y etiquetas que se utilizan para describir el formato y la presentación de los documentos en línea. HTML se compone de una serie de etiquetas o elementos, cada uno con una función específica. Estas etiquetas se utilizan para marcar y estructurar diferentes partes del contenido de una página web, como encabezados, párrafos, imágenes, enlaces, listas, tablas, formularios y otros elementos. Para escribir en la versión actual de html (html 5) debemos colocar la declaración !DOCTYPE html (entre signos de menor y mallor).

Elementos

Son las partes fundamentales que componen una página HTML. Están representados por etiquetas y se utilizan para estructurar y dar formato al contenido. Cada elemento comienza con una etiqueta de apertura etiqueta (entre signos de mayor y menor) y termina con una etiqueta de cierre /etiqueta (entre signos de menor y mayor).

Atributos

Proporcionan información adicional sobre los elementos HTML. Se agregan dentro de las etiquetas de apertura y se componen de un nombre y un valor. Los atributos ayudan a controlar el comportamiento o la apariencia de los elementos. El atributo se escribe dentro de la etiqueta del elemento seguido de un = y “” con el valor del atributo. Puedes incluir varios atributos separados por espacios en la etiqueta de apertura del elemento.
